# AI Coding Agent Training

## Purpose

This workspace contains training materials for technical teams learning to use GitHub Copilot AI agents in Visual Studio Code.

## Current Structure

- `materials/` for learner-facing modules and slide assets
- `labs/` for hands-on exercises, starter state, and solution guidance
- `references/` for shared factual reference documents
- `.github/prompts/` for reusable Copilot prompt files
- `.github/copilot-instructions.md` for workspace-wide guidance
